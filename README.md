#　凸轮控制器
---
  
##　功能简介：
　　根据光电编码器测得的角度信息，控制12个通道的高速电磁阀输出；  
　　用户通过触摸屏可以设置工作参数（输出开启角、关闭角等）；  
　　也可通过六路数字信号输入，对程序运行加以控制；
